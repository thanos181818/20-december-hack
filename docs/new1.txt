**Page: Portal User - Home Page**

**Header/Navigation:**
- Company Logo → Home Page
- Home link → Home Page
- Shop link → Product Listing page
- My Account link → only when logged in, goes to My Account/Orders page
- Authentication: Sign In button (if not logged in), User name display with Sign Out option (if logged in)
- Cart icon → Cart Page

**Body:**
- Displays Home Page content
- Landing page after login/logout/order completion
- Shows promotional/featured products

**User Visibility Rules:**
- My Account menu hidden for guests, visible only when logged in
- Cart accessible for all, checkout requires login

**Navigation Flow:**
- Home Page → Shop Page
- Home Page → Product Detail Page
- Home Page → Cart Page
- Home Page → Sign In Page
- Home Page → My Account Page (logged-in only)

**Page: Shop Page - Product Listing**

**Header:** (same as Home Page)
- Shop link highlighted

**Breadcrumb Navigation:**
- Shows path (All Products / Category / Product Name)
- Each level clickable

**Category Tabs:**
- T-Shirts, Shirts, Kurtas, Formals, Jeans, Hoodies, Sarees, Nightwear
- Click filters products by type

**Left Sidebar Filters:**
- Category: Men, Women, Children
- Size: S, M, L, XL, etc.
- Material: Cotton, Nylon, Others
- Color: multiple selection
- Price Range: slider/selector
- Brand/Seller tag (optional)

**Search & Sort:**
- Search bar for product name/description
- Sort by: Price (Low to High), Price (High to Low)

**Product Grid:**
- Product cards with image, name, description, price
- Click product → Product Detail Page

**Visibility Rules:**
- Only published products shown
- Stock availability affects display/add-to-cart

**Navigation Flow:**
- Shop Page → Product Detail Page
- Shop Page → Cart Page
- Shop Page → Home Page
- Shop Page → My Account (logged-in)
- Shop Page → Checkout Page

**Page: Cart Page**

**Header:** (same as Home Page)
- Cart icon shows item count

**Checkout Progress:**
- Order → Address → Payment (Order step highlighted)

**Cart Items:**
- Product name, price, quantity selector, remove button
- Quantity changes update totals
- Remove button removes item

**Discount Section:**
- Discount code input field with Apply button
- Discount appears as separate line after application

**Price Summary:**
- Subtotal, Taxes, Total payable
- Updates dynamically

**Checkout Action:**
- Checkout button → Address Page (requires login if not logged in)

**Address & Payment Preview:**
- Shows products, subtotal, taxes, total
- Confirm button in final payment step

**Navigation Flow:**
- Cart Page → Product Page
- Cart Page → Address Page
- Cart Page → Payment Page
- Cart Page → Order Confirmation Page
